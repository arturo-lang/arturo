Elegant & Minimal

Clean, expressive syntax that gets out of your way. No semicolons, no braces, no noise. Just clear code that says exactly what it means. Learn the basics in minutes, master the rest naturally.

Fast & Efficient

Optimized bytecode VM delivers solid performance where it matters. Start instantly—no warm-up time, no lengthy compilation. Just run your script and get results.

Truly Portable

Works on Linux, macOS, Windows, FreeBSD. Bundle your scripts into standalone executables—distribute them anywhere without asking users to install anything.

Batteries Included

Web servers, UI toolkit, databases, cryptography, HTTP clients, templating—it's all built in. Need more? Extend with our package ecosystem. Everything ready from day one.

Highly Flexible

No reserved keywords. Code is data, data is code. Redefine operators, create domain-specific languages, shape the language to match your problem—not the other way around.

Made with Love

Open source and community-driven. Started as an experiment, evolved into a tool people actually use. Join us in building something genuinely different and fun to work with.

Get Arturo

Get up and running in seconds.
No fuss. Just code.

copy
curl -sSL https://get.arturo-lang.io | sh

Copy-paste the code above in your terminal and Arturo will be automatically installed.
 Supports Linux, macOS, FreeBSD, and Windows (WSL/Git Bash/MSYS2).
